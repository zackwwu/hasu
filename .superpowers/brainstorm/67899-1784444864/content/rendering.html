3D Isometric Room View

Transform

No 3D engine. Isometric = 2D matrix on Canvas.

// Surface point → screen point
ix = (sx - sy) * cos30° + ox
iy = (sx + sy) * sin30° - sz + oy

// Canvas handles it:
canvas.save();
canvas.transform(matrix);
drawSurface(surface);  // same painters
canvas.restore();

Same tile painting code used for 2D runs through the isometric transform — textures skew correctly, grout lines stay aligned.

Draw Order (painter's algorithm)

1.Floor (farthest)
2.Back walls
3.Side walls
4.Front wall (nearest)

Static isometric angle — no camera rotation. Draw order is deterministic. No z-buffer needed.
