Architecture Overview

5 core modules, data flows left to right

📐 Room Modeler

Define walls & floors
Set dimensions & positions
Assign tile groups

🧱 Tile Library

Define tile groups
Set tile sizes
Capture / import textures

📸 Texture Capture

Camera → edge detect
Perspective correct
Save as tile texture

⬇

⚙️ Layout Engine

Compute optimal tile placement
Symmetry & balance algorithm
Eliminate sliver cuts
Grout alignment (stretch)

🎨 Render Engine

CustomPainter on Canvas
2D elevation views
Isometric 3D views
Dimension labels & cut marks
High-res PNG export

⬇

🖼️ Preview & Export

Interactive 2D view with drag
3D client preview
Export PNG / PDF
Share from device

💾 Local Storage

SQLite via sqflite
All room + tile + layout data
No network needed

Data Flow

Room Modeler ──▶ Layout Engine ──▶ Render Engine ──▶ Preview/Export
(walls, dims)           (tile positions)      (pixels)

Tile Library ──▶ Layout Engine
(tile sizes, textures)

Texture Capture ──▶ Tile Library
(images)

All modules ──▶ Local Storage (SQLite)
(persist everything)
