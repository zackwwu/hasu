How It's Generated

  • Walk all LayoutResult.tiles across all surfaces
  • Filter isCut == true
  • Group by: tileGroup + {w, h} + cutEdge
  • Count per group → "×5"
  • Sort: tile group, then by dimension
  • Full tile counts shown per group header
